acmedigital
===========

Please, run it on mac os x.

**clone**:
	
	git clone git@github.com:gporpino/acmedigital.git
		

**configure**:

	cd acmedigital
	
	./configure
	
**run**:

	ruby execute.rb
	
**simulate**:	
	
	clockwork simulate.rb
	
to test **execute**:
	
	rspec controle_temperatura_spec.rb
